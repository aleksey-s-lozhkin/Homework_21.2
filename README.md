## Веб-сервер для обработки форм контактов
### Описание проекта
Простой HTTP-сервер на Python для обработки HTML-форм контактов. Сервер читает HTML-файл и обрабатывает как GET, так и POST-запросы.

### Основные возможности
Обработка GET-запросов для отображения HTML-формы

Обработка POST-запросов с выводом данных в консоль

Автоматическое добавление сообщения об успешной отправке

Базовая обработка ошибок

### Системные требования
Python версии 3.6 или выше

Стандартный модуль http.server

### Настройка и запуск
#### Установка
Клонируйте репозиторий

Перейдите в директорию проекта

#### Запуск сервера

python main.py
Конфигурация
### Основные настройки находятся в файле main.py:

hostName - адрес сервера (по умолчанию localhost)

serverPort - порт сервера (по умолчанию 8080)

HTML_FILE_PATH - путь к HTML-файлу

### Использование
После запуска сервера вы можете открыть в браузере адрес:

http://localhost:8080
